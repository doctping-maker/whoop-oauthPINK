Privacy Policy

Pink Whoop Assistant is a personal tool used by the account owner to access and review WHOOP data for personal health insights.

What data is accessed

This application may access WHOOP account data such as recovery, sleep, strain, HRV, resting heart rate, workout, and profile information, only after explicit user authorization.

How data is used

The data is used only to generate personal health summaries, trend analysis, and daily/weekly wellness insights for the authorized user.

Data sharing

This application does not sell or share personal WHOOP data with third parties. Data is used only for the private analysis workflows authorized by the user.

Data storage

Authorized tokens and related configuration may be stored locally in a private environment controlled by the user for the purpose of accessing WHOOP data.

User control

The user may revoke access at any time through WHOOP account settings or by deleting locally stored credentials.

Contact

For questions regarding this application, please contact the application owner.
